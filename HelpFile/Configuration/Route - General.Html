Route - General



The general tab on the route configuration is very basic. Here you will define the name of the route (which is whatever you want to call it), and the description.

  • Name: The name of the route.
  • Traffic Type: The traffic type will determine what options are available on the successive configuration tabs. The options are HTTP, HTTPS and Binary. If you are routing raw web-traffic, use HTTP, if its SSL, use HTTPS. For any other type of traffic (for instance: email, FTP, video streaming): choose Binary.
  • Auto Start: The "auto start" checkbox allows you to configure whether the route will auto-start when the service is started.
  • Description: The description of the route.
