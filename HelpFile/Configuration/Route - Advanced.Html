Route - Advanced



Advances settings should only be changed if you have a compelling reason to do so - in fact, it is recommended that you review the service source code before modifying these values.

  • Buffer:
    • Initial Size: The initial size of the communication buffer for each connection.
    • Max Size: When data is flowing into NetProxy faster than downstream systems can consume it, the buffer can grow. This defined the maximum size of the buffer per connection
    • Accept Backlog Size: The size of the incomming connection backlog queue. https://msdn.microsoft.com/en-us/library/system.net.sockets.socket.listen.aspx
  • Misc.:
    • Encryption Initilization Timeout (ms): For proxy connections, this is the amount of time that NetProxy will wait for the negotiation to complete before disconnecting the connection.
    • Spin-Lock Count: For proxy connections, this is the number of loops to perform to cause a limited delay when data is received before encryption is negotiated. This is a performance setting and will not affect overall functionality.
    • Sticky Session Cache Expiration (s): The number of seconds that any incomming connections are bound to a given endpoint before falling back to the balancing rule.
